# 🏨 Hotel Revenue, Cancellations & Customer Behavior Analysis

## 📌 Опис проєкту

Цей проєкт присвячений аналізу даних готельних бронювань з метою дослідження **поведінки клієнтів, попиту, доходів та скасувань**.

Мета проєкту - отримати бізнес-інсайти, які допоможуть:

- зменшити рівень скасувань;
- підвищити дохід;
- краще зрозуміти поведінку клієнтів.

Аналіз охоплює:

- динаміку бронювань;
- поведінку клієнтів (lead time, is_repeated_guest);
- дохід та потенційні втрати;
- вплив скасувань на бізнес.

---

## 🎯 Цілі аналізу

- Дослідити попит на готельні бронювання.
- Виявити фактори, що впливають на скасування.
- Оцінити фінансову ефективність.
- Проаналізувати поведінку клієнтів.
- Оцінити потенційні втрати доходу через скасування.

---

## 🧹 Обробка даних

- Датасет перевірено на пропущені значення та оброблено їх залежно від типу змінної.
- Оптимізовано типи даних для окремих колонок.
- Створено нові ознаки:
  - `total_price` - загальна вартість бронювання;
  - `total_guests` - загальна кількість гостей;
  - `arrival_date` - дата прибуття.
- Перевірено дані на дублікати та логічні невідповідності.
- Видалено некоректні спостереження та викиди.
- Підготовлено вибірки для дослідницького аналізу даних (EDA) та перевірки статистичних гіпотез.

## 📈 Основні інсайти

- Гості бронюють готелі в середньому за **105 днів до заїзду**, що свідчить про довгий цикл прийняття рішення.
- Рівень скасувань є значним і призводить до **суттєвих втрат потенційного доходу (~1.5 млн)**.
- **Лояльність клієнтів є низькою** - лише **3.21% гостей здійснюють повторні бронювання**.
- Середня тривалість перебування становить **3-4 дні**, що характерно для коротких подорожей.
- Скасування суттєво впливають на стабільність доходу готелів.

Аналіз показав, що ключовими викликами для готельного бізнесу є високий рівень скасувань та низька частка повторних гостей. Зменшення кількості скасованих бронювань і підвищення лояльності клієнтів можуть суттєво покращити фінансові результати та стабільність доходів.
